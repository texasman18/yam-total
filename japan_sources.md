## 데이터 소스

### 1. OpenStreetMap (이번 버전의 유일한 번들 소스)
- 추출일: 2026-07-16, Overpass API (`amenity=toilets`, node+way)
- 라이선스: **ODbL — © OpenStreetMap contributors** (앱 내 "데이터 출처" 화면에 고지)
- 정제 규칙: `access=private|customers` 제외, 도시 내 30m 이내 중복 제거(태그 많은 쪽 유지), 좌표 6자리 반올림
- 필드: 이름(`name:ko` 우선, 없으면 `name`, 없으면 公衆トイレ), 운영시간(opening_hours), 설비(f: wc=휠체어/ba=기저귀교환대/os=오스토메이트)

### 2. 일본 지자체 오픈데이터 (후속 예정)
- 자치체표준오픈데이터셋 "公衆トイレ一覧"(CC BY) 병합은 **후속 작업으로 예정** — 병합 시 배리어프리·오스토메이트 등 설비 필드가 보강됨. 스키마의 `s:"city"` 값이 예약돼 있음.

### 3. Google Places (런타임 보완 — 번들에 미포함)
- 앱 실행 중 `searchByText`(公衆トイレ/トイレ, language=ko)로 보완. 앱 내 "일부 결과는 Google 검색 결과를 포함합니다" 고지.
