- 파이프라인: `pipeline/fetch_moenv.py`(환경부 수집) + `pipeline/fetch_osm.py`(OSM 수집)

## 데이터 소스

### 1. 대만 환경부 「全國公廁建檔資料」 (주력 소스)
- 출처: 環境部 環境管理署, 데이터셋 ID **FAC_P_07** (data.moenv.gov.tw)
- 수집일: 2026-07-16, offset 페이지네이션 전량 수집 (limit=1000 × 46회) — 원본 **45,718건**
- API 키: **data.gov.tw가 공식 배포하는 다운로드 URL에 내장된 공개 키**(`e75b1660-…`)를 사용.
  익명 접근용으로 정부가 노출한 키이나, 장기 운영 시 **사용자 개인 키 발급·교체 권장** (data.moenv.gov.tw 회원가입 무료)
- 라이선스: **政府資料開放授權條款 v1 (OGDL-Taiwan-1.0, CC BY 4.0 호환)** — 앱 내 "데이터 출처" 화면에 고지
- 수집 시 참고: data.moenv.gov.tw 인증서가 Python 3.14 OpenSSL 검증("Missing Subject Key Identifier")에 걸려 fetch_moenv.py는 urllib 대신 curl을 사용
- **건물 단위 그룹핑**: 원본은 물리적 화장실 1곳이 남廁/여廁/친자廁/무장애廁·층별 개별 레코드
  (예: "太平區公所4F-男廁", "太平區公所1F-親子廁所" 이 같은 좌표) —
  name에서 성별·층 접미사(`-男廁`, `-女廁`, `-親子廁所`, `-無障礙廁所`, `4F`, `B1`, `二樓` 등)를 제거한
  기본명+주소로 그룹핑해 1레코드로 병합. **45,718건 → 19,617건** (좌표 결측/0/범위밖 59건 스킵)
  - diaper(기저귀대) 그룹 내 max>0 → `f.ba=1`
  - 그룹에 無障礙 레코드 존재 → `f.wc=1`
  - grade는 그룹 내 최고 등급 채택 (特優級>優等級>普通級>改善級>不合格)

### 2. OpenStreetMap (보조 소스 — 등산로 등 환경부 미등재 결측 보충)
- 추출일: 2026-07-16, Overpass API (`amenity=toilets`, node+way), `area["ISO3166-1"="TW"]` 전국 단일 쿼리
- 라이선스: **ODbL — © OpenStreetMap contributors** (앱 내 고지)
- 정제 규칙: `access=private|customers` 제외, 원본 5,741건 → 정제 후 5,652건
- **병합 우선순위: 환경부 > OSM** — 30m 이내 중복 시 환경부 레코드 유지, OSM의 h(운영시간)만 보충(27건).
  OSM 단독 지점 4,050건은 유지.

### 3. Google Places (런타임 보완 — 번들에 미포함)
- 앱 실행 중 `searchByText`(公廁/廁所, language=ko)로 보완. 앱 내 "일부 결과는 Google 검색 결과를 포함합니다" 고지.

## 한글화 후처리 (translate.py)
- ZH_KO 사전(약 330개 항목, 긴 단어 우선 매칭): 대만 22개 현·시, 주요 관광지·행정구,
  화장실/공공시설 용어, 환경부 빈출 시설 용어(區公所→구청, 里民活動中心→주민센터, 統一超商→세븐일레븐,
  台灣中油→타이완중유, 衛生所→보건소 등), 유통 브랜드
- 사전 미등록 한자는 `hanja` 패키지로 한국 한자음 변환 폴백
- 결과: 전체 17,561건 중 17,547건에 k(한글명) 생성, **한글화율 100%**(표시명에 한자 미잔존)
