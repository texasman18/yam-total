- 타깃: ① 방일 한국인 ② 방대만 한국인 ③ 방한 외국인

## 1. 전략 요약 (한 문장)

> "화장실이 급했던 경험담"은 여행 커뮤니티에서 가장 잘 도는 이야기다 — 경험담 콘텐츠에 앱을 자연스럽게 얹어, 검색(SEO·ASO)과 커뮤니티 입소문 두 축으로 무료 확산한다.

## 2. 채널별 실행 플랜

### 2-1. ASO (앱스토어 최적화) — 최우선, 1회 작업으로 상시 효과
- [ ] App Store 제목·서브타이틀·키워드 100자를 검색어 문서 §4-1로 교체 (빌드 불필요, 메타데이터만)
- [ ] Google Play 짧은/긴 설명 갱신 (§4-2)
- [ ] 스크린샷 첫 장을 "3개국 자동 전환" 비주얼로 교체 — 첫 장이 전환율의 대부분을 결정
- [ ] 일본·대만 국가 스토어 가용성 확인 (기존 체크리스트 ④와 동일)
- 주기: 배포 시 1회 + 분기 1회 키워드 재점검

### 2-2. 네이버 블로그·카페 (방일·방대만 한국인의 주 서식지) ★핵심
- 블로그: 콘텐츠 제목 뱅크(검색어 문서 §5)로 주 1편 × 8주. 구조는 "경험담 → 정보(화장실 문화·팁) → 앱은 마지막에 한 줄". 광고 티가 나면 역효과.
- 카페: 네일동(네이버 일본여행 동호회), 대만은 즐거워(대만 여행 카페) 등 대형 여행 카페의 **질문 글에 답변**으로 진입 — "화장실 어디서 찾아요?" 류 질문에 경험 기반 답변 + 앱 언급. 홍보 글 직행은 강퇴 사유이므로 금지.
- 지식iN: "일본 화장실", "대만 화장실 휴지" 질문 답변 누적.

### 2-3. 쇼츠·릴스 (유튜브 쇼츠 / 인스타 릴스 / 틱톡) — 동일 소스 3플랫폼 재활용
- 포맷: 15~30초. "일본에서 급똥 → 앱 켬 → 3초 만에 지도" 화면 녹화 + 자막. 기존 보유한 9:16 변환 워크플로우 재활용.
- 소재 6종: 신주쿠 실사용 / 지우펀 실사용 / 자동 전환 순간(비행기 착륙 컨셉) / 대만 위생등급 뱃지 소개 / 방한 외국인용 영어 버전 / "편의점 화장실 안 돼요" 정보형
- 주기: 주 2편 × 6주 = 12편. 계정은 기존 개인/테크셀 계정 활용.

### 2-4. 해외 채널 (방한 외국인)
- Reddit: r/koreatravel, r/seoul의 화장실·여행팁 질문에 답변 (셀프 프로모션 규칙 준수 — 답변 10개당 앱 언급 1개 수준)
- 트립어드바이저 서울 포럼, Korea travel Facebook 그룹 동일 방식
- 웹 랜딩 영어 meta 최적화 (검색어 문서 §3, §4-3)

### 2-5. 커뮤니티 단발 이벤트성
- 클리앙·더쿠·뽐뿌 여행 게시판: "일본/대만 여행 갈 때 쓰라고 만든 무료 화장실 앱" 개발기 형식 1회 — 개발자 직접 후기 형식은 커뮤니티 반응이 좋고 무료
- 오픈채팅(일본여행·대만여행 방): 프로필/공지 활용은 방장 협의 후

### 2-6. 제휴 (비용 0)
- 일본·대만 여행 유튜버/블로거 5~10명에게 DM: "무료 앱이라 협찬비는 없지만, 구독자에게 유용할 것" — 소형(1~5만) 채널 위주로 회신율이 높음
- 여행 준비물 체크리스트 블로거에게 "필수 앱" 목록 등재 요청

## 3. 8주 실행 캘린더 (8월 배포 가정)

| 주차 | 작업 |
|---|---|
| W1 (배포 주) | ASO 전체 갱신 + 웹 SEO 메타 + 블로그 1편(일본) |
| W2 | 쇼츠 2편(신주쿠·자동전환) + 카페 답변 시작(주 3회 상시) |
| W3 | 블로그 1편(대만 휴지 이슈) + 쇼츠 2편 |
| W4 | 커뮤니티 개발기 1회(클리앙) + 블로그 1편 |
| W5 | Reddit·트립어드바이저 답변 시작 + 쇼츠 2편(영어 1) |
| W6 | 유튜버 DM 10건 + 블로그 1편 |
| W7 | 쇼츠 2편 + 지식iN 몰아서 10건 |
| W8 | 성과 리뷰(§4) 후 잘 되는 채널에 집중 재배분 |

## 4. 측정 (무료 도구만)
- 스토어: App Store Connect 노출·전환, Play Console 검색어 리포트
- 웹: Vercel Analytics(무료 티어) 또는 Cloudflare — 페이지뷰·유입국
- 앱 내: 계측 없음(현행 유지) — 해외 모드 진입율 KPI는 백로그(기획서 §7)와 동일
- 성공 기준(8주): 스토어 페이지 조회 주간 +50%, 블로그 누적 1만 뷰, 쇼츠 1편 이상 1만 재생

## 5. 금지·주의
- 커뮤니티 직접 홍보 글 도배 금지 (강퇴 + 브랜드 이미지 훼손)
- 리뷰 조작·품앗이 금지 (스토어 정책 위반)
- 쇼츠에 지도 화면 쓸 때 Google/Kakao 로고 크레딧 화면 유지 (라이선스)
- 모든 발행물은 사용자(이석진) 최종 승인 후 게시 — 승인 전 발송·게시 금지 원칙과 동일
